# Background Music Setup

## How to Add Background Music

The website is set up with a professional background music controller. To add your music:

### Step 1: Get Your Music File
- Find a calm, professional background music track
- Recommended: Instrumental, ambient, or soft acoustic music
- Format: MP3 (best compatibility)
- Length: 2-5 minutes (it will loop automatically)

### Step 2: Add the Music File
1. Save your music file as: `background-music.mp3`
2. Place it in the `files/` folder
3. The full path should be: `files/background-music.mp3`
