Memory Match Game

Welcome to my interactive memory match game! This engaging game features Clash Royale character cards that you'll need to match by memory. Test your concentration and memory skills as you flip cards to find matching pairs.

How to Play: Flip two tiles to find matching icons. Use the keyboard: arrow keys to navigate tiles, Enter or Space to flip. Click Start to begin the timer and background music.

Press R to restart the game at any time. The game tracks your moves and time, so challenge yourself to complete it as quickly as possible with the fewest moves!

Features: This game includes background music, smooth animations, keyboard navigation for accessibility, and a professional hunting-themed design that matches my portfolio aesthetic.
